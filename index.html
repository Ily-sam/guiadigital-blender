Guía Digital Didáctica de Blender

Proyecto de tesis con aprendizaje interactivo y visualización en Realidad Virtual

Ver Proyecto

📘 Información del Proyecto

Este proyecto consiste en el desarrollo de una guía digital didáctica para el aprendizaje del software Blender, incluyendo prácticas guiadas y visualización en realidad virtual (VR).

Objetivo General

Crear una guía educativa que enseñe los fundamentos con su aplicación práctica en Blender 3D, incorporando herramientas de realidad virtual para mejorar el aprendizaje interactivo y las habilidades digitales de cada uno de los estudiantes.

Objetivos Específicos

  • Describir las funciones y herramientas básicas necesarias
  • Implementar elementos de realidad virtual
  • Crear modelos tridimensionales básicos
  • Fomentar el aprendizaje autónomo
